Commands
Install
$ npm install -g @angular/cli@15

CLI Commands

Create new project
$ ng new <prj-name>

Run the application
$ ng serve
$ ng test

Add Component
$ ng g component <component-name>

Add Service
$ng g service shared/search/search
where 'shared/search/search is the path to the service 'search'.
